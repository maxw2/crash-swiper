# crash-swiper

使用原生js编写的轮播图组件

## Installation


```bash
npm i crash-swiper
```

## Usage

```python
import CSwiper from 'crash-swiper'

let swiper = new CSwiper(el,option)

```

## option

| key     | value                         | type |
|:-----   | :---------------------------  |:-----|
|imgList  | ['./img1.jpg','./img2.jpg']   |array |
|type     | {rows:number,cols:number} || 1|object|
